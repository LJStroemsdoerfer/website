Python API

Overview

The till package allows you to create Docker images automatically from your flask, or FastAPI API. You will only need to provide your API file and any directory you need and till will create the images, build and test them, and ship the image to any remote registry you want.

Usage

Installation

The till package is hosted on PyPi, however, can only be accessed with a secret key. At subscription start, you will receive this key from us.

pip install till --auth your_key

Code Example

# load library
from till import container

# init container class
my_container = container(api = "your_api_file.py",
			 dependencies = "your_dir/",
			 port = "5000")

# create the docker image
my_container.build()

# run the docker container
my_container.run()

# push the docker container
my_container.push(registry = "my-regisry:443/my_project",
                  auth = {"password" : password, "user" : user})
