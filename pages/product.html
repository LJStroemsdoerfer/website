Effortless Kubernetes Deployments

Python API

Our Python API enables a stunningly fast construction of Docker containers. With a full integration into the Flask API framework, Pythonists can construct, build, run and deploy containers without leaving Python.

Data Scientists can build their API in Flask and till will take care of catching the necessary libraries and file resources.

In addition to the build() and run() methods, you have a deploy() method, which allows you to ship your API to any remote registry.

till is available on PyPi. So you can download the library as every other Python library, using: pip install till

To unlock your installation, you need to activate your license ones in your python terminal, using: till.activate_license(key = 'your_key')

R API

Our R API allows you to construct Docker containers directly from RStudio. There is no need to take any detours or extra effort. Our package integrates perfectly withe the most promiment R API framework plumbeR. R Data Scientists can simply build their plumbeR API as they are used.

till takes care of all the heavy lifting. It checks all necessary package dependencies, as well as file resources that are needed to run the API.

We comply with the standard R interface, instead of running plumbeR run commands, you can use the till functions to get your API containerized.

Similar to the Python API you can not only build and run containers, but also deploy them to any remote location and all of this can be done, from your favorite R IDE.

To install till you can simply use your standard CRAN mirror and run the typical installation command: install.packages('till').

till is a licensed product, thus you will have to acivate your license ones. this can be done using: till::activate_license(key = 'your_key').
